# Ritik Gupta – Portfolio Website

This repository contains the source code for my personal portfolio website, showcasing my skills, projects, and resume as a web developer.

🔗 **Live Website:** https://ritikgupta8.in  
📄 **Resume:** Available on the website

---

## 📌 About

This is a clean, responsive, single-page portfolio website built to present:
- Who I am
- What I build
- The technologies I work with
- My projects and experience
- Ways to contact me

The site is fully static and optimized for fast loading and easy maintenance.

---

## 🛠 Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Layout, styling, responsiveness
- **JavaScript (Vanilla)** – Interactivity and dynamic behavior
- **GitHub Pages** – Hosting
- **Custom Domain (.in)** – Secured with HTTPS
